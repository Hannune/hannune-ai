# Portfolio Website

Personal portfolio website for AI/ML Engineer showcasing projects, skills, and services.

**Live Site:** https://hannune.github.io/hannune-ai/

## Features

- Dynamic content loading from JSON
- Magnetic cursor interactions
- Floating background animations
- Fully responsive design
- Contact form integration with Formspree

## Tech Stack

- HTML5
- CSS3 (with animations)
- Vanilla JavaScript
- GitHub Pages

## Local Development

1. Clone the repository
2. Open `index.html` in a browser
3. Edit `content.json` to update content

## Updating Content

All content is managed through `content.json`. Simply edit this file and push to update the live site.

```bash
git add content.json
git commit -m "Update content"
git push
```

The site will automatically redeploy.
